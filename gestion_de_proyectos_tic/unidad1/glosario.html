Glosario

Case

Siglas en inglés para: Computer Aided Software Engineering, Ingeniería de Software Asistida por Computadora

Casos de uso

Conjunto de pasos ordenados, o incluso actividades que secuencialmente se realizan cuando se ejecuta un proceso.

Ingeniería de requerimientos

También llamada ingeniería de requisitos. Se relaciona con todas las actividades que se realizan para determinar las necesidades de un sistema.

Stakeholders

En español quiere decir “personas involucradas”. Se trata de un término en inglés que normalmente se emplea para definir a las personas que de una forma u otra resultan afectados, o pueden afectar las actividades de un proyecto, o de una empresa.

UML

Siglas en inglés para Unified Modeling Language, el lenguaje unificado de modelado.
